Two layers per Android spec: the node + line ride the foreground, the board gradient is the background, so launchers can mask, parallax, and theme it. The node stays legible and centred at every size; below 48px it drops the line and keeps just the glowing node.

✕ Don't
  • Recolour the node to brand-blue
  • Add a drop shadow to the wordmark
  • Stretch, outline, or rotate the mark
  • Place on busy photography
